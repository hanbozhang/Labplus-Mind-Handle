## 命令规则

| 命令                 | 说明                   | 示例                             |
| -------------------- | ---------------------- | -------------------------------- |
| LX###LY###RX###RY### | 传输两个摇杆XY轴的数据 | LX0LY0RX0RY0表示当前摇杆全部居中 |
| takeoff              | 自动起飞               | takeoff                          |
| land                 | 自动降落               |                                  |
| emergency            | 停止电机转动           |                                  |
| throwfly             | 抛飞                   |                                  |
| motoron              | 进入起桨模式           |                                  |
| motoroff             | 退出起浆模式           |                                  |
| rc xx xx xx xx       | 摇杆控制               | rc 50 50 50 50                   |
|                      |                        |                                  |
